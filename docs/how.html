How the Internet Works

In this section, we explore how does the internet actually work? What is the underlying infrastructure and technology that connect us all? This website will provide some of the most basic information of the concepts included in this topic. It will not go into great detail but hopefully, it will provide you with a simple understanding of the internet to this day.

Domain Name & ICANN

The Internet is a giant network of computers connected to each other through a global network of cables. Domain names were invented to help make the browsing experience easier.

IP Address, Packets & Routing

With shared web hosting there may be hundreds or even thousands of websites on the same server. A unique IP address offers a layer of protection and greater security and reliability.

Domain Name System

The Domain Name System (DNS) is the phonebook of the Internet. Humans access information online through domain names, like nytimes.com or espn.com.

HTTP and HTTPS Protocol

HyperText Transfer Protocol Secure (HTTPS) is the secure version of HTTP, the protocol over which data is sent between your browser and the website that you are connected to.

W3C and HTML/CSS

HTML (The Hypertext Markup Language) and CSS (Cascading Style Sheets) are two of the core technologies for building Web pages.
